Este documento registra la evolución técnica y funcional de KitPOP PWA.

# v0.0.1 – Fundación del proyecto

## Arquitectura

- Proyecto creado con React + Vite.
- React Router configurado.
- Supabase inicializado.
- Variables de entorno configuradas.
- Estructura base del proyecto creada.

## Componentes

- Layout.
- Topbar.
- Sidebar (estructura inicial).
- Footer.

## Home

- Hero migrado desde el HTML oficial.
- CategoryGrid.
- CategoryCard.
- Categorías separadas en `src/data/categories.js`.

## Metodología

- El HTML oficial pasa a ser la fuente maestra del diseño.
- Se establece una arquitectura basada en componentes reutilizables.
- Se adopta la metodología de "archivo completo" para todas las modificaciones.
